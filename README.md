# fif-changeScore

可以修改fif口语的成绩

需要配合Fiddler食用

## 使用指南

打开Fiddler  抓包配置请自行搜查配置

Rules-> Customize Rules 

将本仓库中的fif.js复制替换上去即可（或者只修改OnbeforeRequest方法）



## 注意事项

本脚本编写初衷只是因为AI评分过于迷惑+分数会算到平时分中所以想要提高fif的成绩，该读还是要读的，排行榜上是能听到别人的音频的/(ㄒoㄒ)/~~
